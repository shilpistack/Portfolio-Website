# 🌐 Personal Portfolio Website

##  Overview

This is my personal portfolio website built using HTML, CSS, and JavaScript. It showcases my skills, projects, and contact information in a clean and responsive design. The purpose of this project is to present my work professionally and create an online presence for internship and job opportunities.

##  Features

* Responsive and modern UI design
* Navigation between multiple pages (Home, About, Skills, Contact)
* Clean and structured layout
* Smooth user experience
* Beginner-friendly project structure

##  Project Structure

* **index.html** – Home page
* **about.html** – About me section
* **skills.html** – Skills and technologies
* **contact.html** – Contact details
* **CSS files** – Styling for each page
* **JavaScript file** – Basic interactivity
* **images folder** – Contains all images used in the project

##  Technologies Used

* HTML5
* CSS3
* JavaScript

##  Purpose

This project is created to:

* Practice front-end development skills
* Build a professional portfolio
* Showcase my learning and projects
* Prepare for internships and job opportunities
